# TrafiBoard

TrafiBoard is an innovative `Reverse-Proxy-to-Dashboard` application that aggregates multiple Reverse Proxies to parse their data and generate dynamic dashboards that are always up to date with your infrastructure! 

No more bookmarks or manually managed dashboards with broken links!

## Who is it for?
- Tired of mantaining dashboards and bookmarks of your services?
- Do you use Traefik as reverse proxy for your Docker containers?
- Maybe Nginx Proxy Manager?

If you answered YES to any of the above then TrafiBoard may be the tool for you.

TrafiBoard automagically generates a dashboard/index of services from reverse proxies (even multiple).

## Features
- Supported Reverse Proxies:
  - [Traefik](https://traefik.io/traefik/)
  - [Nginx Proxy Manager](https://nginxproxymanager.com)
- Aggregate and index multiple proxies
- Deduplication of common routes
